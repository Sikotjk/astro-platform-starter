# Gebündelte Schriftarten

`DejaVuSans.ttf` / `DejaVuSans-Bold.ttf` werden zur PDF-Erzeugung des
Zoll-Manifests eingebettet. DejaVu deckt Lateinisch **und Kyrillisch** ab
(nötig für Russisch/Tadschikisch).

**Lizenz:** DejaVu Fonts (frei, redistribuierbar — abgeleitet von Bitstream
Vera / Arev). Siehe https://dejavu-fonts.github.io/License.html
